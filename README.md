# TaskFlow

TaskFlow is a Django-based task management system that allows users to manage projects, tasks, and comments.

This project was created as a learning project to demonstrate backend development with Django, authentication, and project collaboration features.

---

## Features

- User registration and authentication
- Project creation and management
- Task management inside projects
- Project member management
- Comments on tasks
- Simple dashboard interface

---

## Tech Stack

- Python
- Django
- SQLite
- HTML
- Bootstrap
- Git / GitHub

## Installation
Clone the repository:
git clone https://github.com/Realsnubs/
tasktlow.git

Go to the project directory:
cd taskflow

Create virtual environment:
python -m venv venv

Activate virtual environment:
Windows
venv\Scripts\activate

Install dependencies:
pip install -r requirements.txt

Apply migrations:
python manage.py migrate

Run development server:
python manage.py runserver

Open in browser:
http://127.0.0.1:8000
